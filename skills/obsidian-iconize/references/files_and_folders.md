## Change individual icon color ​

**URL:** https://florianwoelki.github.io/obsidian-iconize/files-and-folders/individual-icon-color.html

**Contents:**
- Change individual icon color ​

If you would like, you can also set an individual icon color for your icon, that is set to a file or folder.

For that you simply need to Right click on your file/folder which contains an icon > Change color of icon. Within the next modal, you can set or reset the color of the icon. Resetting means, setting the icon color to be the default editor color.

## Icon in Tabs ​

**URL:** https://florianwoelki.github.io/obsidian-iconize/files-and-folders/icon-tabs.html

**Contents:**
- Icon in Tabs ​

Make sure, that you've enabled the setting which is related to activating icons in tabs. Please refer to the Settings documentation for more information about this and other options.

Tabs can have icons next to the title of the opened file. To add an icon to a tab, you just need to add an icon to a file and the tab will automatically show it.

## Icon before file or folder name ​

**URL:** https://florianwoelki.github.io/obsidian-iconize/files-and-folders/icon-before-file-or-folder.html

**Contents:**
- Icon before file or folder name ​

It is pretty easy to set a icon before a file or folder name.

You simply need to Right Click a file or folder > Change Icon (keyboard shortcut: Cmd/Ctrl-Shift-j) and then select the icon you would like to set for this specific file or folder. Be aware that this has the highest priority and overrides every other set icon for this file or folder.
