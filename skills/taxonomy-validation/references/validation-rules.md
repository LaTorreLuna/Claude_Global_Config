# ANSI/NISO Z39.19 Validation Checklist

## Structural Rules
- [ ] Single root or 3-7 top-level concepts
- [ ] Depth within recommended range (5-6 levels optimal, warn at 7+)
- [ ] Balanced tree structure
- [ ] Consistent granularity at each level
- [ ] No orphaned concepts (except root and leaves)
- [ ] No circular references

## Relationship Rules
- [ ] BT/NT relationships are reciprocal
- [ ] "Is a kind of" test passes for all BT/NT
- [ ] RT relationships are symmetric
- [ ] No mixing of relationship types

## Term Quality
- [ ] Preferred labels are unique
- [ ] Alternative labels captured for synonyms
- [ ] Scope notes where needed
- [ ] Consistent terminology (singular/plural, capitalization)
- [ ] No undefined jargon
