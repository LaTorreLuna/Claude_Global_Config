# Tag Migration Checklist

## Pre-Migration
- [ ] Git commit with tag "pre-taxonomy-migration"
- [ ] Backup vault
- [ ] Tag inventory completed
- [ ] Target taxonomy designed and approved
- [ ] Migration mapping table created

## Pilot Phase
- [ ] Pilot files selected (10-20 representative files)
- [ ] Pilot branch created
- [ ] Automatic migrations executed on pilot
- [ ] Pilot results validated
- [ ] Lessons learned documented

## Full Migration
- [ ] Final backup created
- [ ] Automatic migrations executed
- [ ] Semi-automatic cases reviewed and processed
- [ ] Manual cases handled
- [ ] Migration scripts logged

## Post-Migration
- [ ] Validation script executed
- [ ] Statistical comparison completed
- [ ] User acceptance testing passed
- [ ] Migration documentation created
- [ ] Training materials prepared
- [ ] Users notified of changes

## Rollback (if needed)
- [ ] Revert to pre-taxonomy-migration git tag
- [ ] Validate rollback success
- [ ] Document failure reasons
- [ ] Plan remediation
