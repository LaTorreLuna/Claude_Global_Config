# Migration Scripts

This directory contains Python scripts for tag migration:

## `analyze_tags.py`
Extract and analyze current tag usage from Obsidian vault.
- Tag inventory with usage counts
- Co-occurrence analysis
- Synonym detection

## `generate_migration_plan.py`
Generate migration scripts from tag mapping table.
- Creates automatic migration script
- Generates semi-automatic candidates list
- Identifies manual review cases

## `validate_migration.py`
Validate post-migration quality and consistency.
- Orphan detection
- Consistency checks
- Statistical reports

**Note**: Scripts to be implemented based on project needs.
